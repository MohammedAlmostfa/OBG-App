# OPG (Old but Gold)

OPG is a platform for buying and selling used tools and equipment. Users can list their items, browse other products, filter and search efficiently, and submit purchase requests. The app aims to provide a smooth, secure, and scalable experience.

---

## 🛠 Features

- Product Listing: Users can add their used products with images and detailed descriptions.  
- Product Browsing: Easy navigation through products with search and filtering options.  
- Purchase Requests: Users can submit purchase requests. Currently, communication happens outside the app, with plans to enable in-app messaging in the future.  
- Secure File Uploads: Images and files are uploaded safely.  
- Advanced Role & Permission System: Implemented with Spatie Role & Permission.  
- High Performance & Scalability:  
  - Flexible database structure  
  - Caching and indexing for improved performance  
  - Clean, maintainable code following SOLID and DRY principles  
  - RESTful API design with Service & Repository Pattern  
- JWT Authentication: Ensures secure login and data protection.  

---

## 🖥 Technologies Used

- Backend: [Laravel](https://laravel.com)  
- Frontend: [Flutter](https://flutter.dev)  
- Database: MySQL / Flexible DB  
- Authentication: JWT  
- Roles & Permissions: Spatie  
- Best Practices: SOLID, DRY, Clean Code, RESTful API, Service Repository Pattern
