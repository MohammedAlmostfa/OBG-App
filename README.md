# OPG - Old but Gold

PG (Old but Gold) is a marketplace application designed for buying and selling used tools and equipment.  
The platform allows users to list products they want to sell while other users can browse, search, filter, and submit purchase requests.

The main idea behind the application is to give used items a second life by creating a simple and accessible marketplace.

---

## 📱 Project Overview

PG acts as a digital marketplace where users can:

- List products they want to sell
- Browse products listed by other users
- Search for specific items
- Filter products based on categories
- View detailed product information
- Submit purchase requests

Currently, communication between buyers and sellers happens outside the application, but a built-in chat system is planned for future releases.

---

# 🎓 University Admission Management Application

In addition to the marketplace project, the application also includes a University Admission Management System available on Google Play.

The system helps students:

- Explore universities
- View available majors and specializations
- Prepare admission preferences
- Organize and manage their application wish list

This helps students make informed decisions when applying to universities.
